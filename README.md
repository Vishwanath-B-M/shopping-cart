# Shopping Cart Website

A modern and responsive Shopping Cart Web Application built using React.js, Redux Toolkit, and Tailwind CSS. The application allows users to browse products, search items, manage cart functionality, and experience a smooth responsive UI.

---

# Features

* Responsive design for mobile and desktop devices
* Product listing with dynamic rendering
* Search functionality for products
* Shopping cart management using Redux Toolkit
* Add to cart and remove from cart functionality
* Authentication token handling using localStorage
* Loading and error handling states
* Reusable React components
* Fast UI styling with Tailwind CSS

# Installation

Clone the repository:

```bash
git clone <https://github.com/Vishwanath-B-M/shopping-cart/tree/main>
```

Move into the project folder:

```bash
cd shopping-cart
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

# Build for Production

```bash
npm run build
```
